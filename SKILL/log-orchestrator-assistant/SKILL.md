# 日志编排技能

## 职责边界

1. 只负责流程编排和质量守卫。
2. 默认链路：`log-filter-assistant -> source-correlation-assistant -> crisp-l-report-assistant`。
3. 用户只要某一步时，执行对应技能，不强制跑全链路。

## 编排策略

1. 用户说“先看样本/缩小范围”：优先筛选技能。
2. 用户说“定位代码/找根因”：优先源码关联技能。
3. 用户说“出报告/落盘”：优先报告技能（可一键）。

## 质量守卫

1. 结论必须有证据锚点。
2. 样本不足必须先提示风险，再给条件化结论。
3. 输出必须全中文，并遵循 CRISP-L 结构（当用户要报告时）。
